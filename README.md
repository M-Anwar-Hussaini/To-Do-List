### Configuring WebPack

1. Initialize package.json:

```sh
npm init y
```

2. Install webpack:

```sh
npm install webpack webpack-cli --save-dev
```

3. Install CSS style loader:

```sh
npm install --save-dev style-loader css-loader
```
